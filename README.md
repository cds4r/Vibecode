# Vibecode Player

Личный музыкальный веб-плеер в стиле Spotify. Загружай **свою** музыку, собирай плейлисты и слушай без ограничений. Работает полностью в браузере — вся музыка хранится локально на твоём устройстве и никуда не отправляется.

## Возможности

- **Своя медиатека** — добавляй аудиофайлы перетаскиванием или через выбор файлов (MP3, FLAC, WAV, OGG, M4A и др.), либо по прямой ссылке на аудио.
- **Локальное хранилище** — треки сохраняются в браузере через IndexedDB и остаются на месте после перезагрузки.
- **Плейлисты** — создавай, наполняй и удаляй; добавляй треки через меню «…» или правый клик.
- **Полноценный плеер** — play/pause, следующий/предыдущий, перемотка, громкость, mute, перемешивание и повтор (всех/одного).
- **Избранное**, поиск по медиатеке, обложки-градиенты, интеграция с Media Session (управление с локскрина/наушников).
- **Горячие клавиши**: `Space` — play/pause, `Alt+→` / `Alt+←` — следующий/предыдущий.
- Адаптивный тёмный интерфейс — от десктопа до телефона.

## Как пользоваться

1. Открой сайт и нажми **«Добавить музыку»**.
2. Перетащи свои аудиофайлы в окно (или выбери их), либо вставь прямую ссылку на аудио.
3. Дважды кликни по треку (или наведи и нажми ▶) — поехали.

## Про источники вроде Яндекс.Музыки

Плеер воспроизводит **твои файлы** и аудио по прямым ссылкам, на которые у тебя есть права. Встроенного скачивания музыки из защищённых каталогов стриминговых сервисов (Яндекс.Музыка, Spotify и т. п.) здесь нет — это нарушало бы их правила и авторские права.

## Запуск локально

```bash
python3 -m http.server 8000
# затем открой http://localhost:8000
```

Данные хранятся в IndexedDB конкретного браузера/домена: на другом устройстве или в другом браузере медиатека будет своя.
