# Vibecode

Современный адаптивный лендинг для вымышленной платформы **Vibecode** — «создавай на скорости мысли».

Сделан как быстрый статический сайт без сборки: чистый HTML, CSS и немного ванильного JavaScript. Открывается напрямую в браузере.

## Возможности

- Адаптивная вёрстка (десктоп, планшет, мобильные)
- Светлая / тёмная тема с сохранением выбора в `localStorage`
- Анимации появления секций при скролле (Intersection Observer)
- Анимированные счётчики статистики
- Мобильное меню-бургер
- Аккордеон FAQ (открыт один пункт за раз)
- Валидация формы подписки
- Плавный скролл, кнопка «наверх», декоративный фон
- Учёт `prefers-reduced-motion` и базовая доступность (skip-link, aria, фокус-стили)

## Структура

```
.
├── index.html
├── assets/
│   ├── css/styles.css
│   ├── js/main.js
│   └── img/favicon.svg
└── README.md
```

## Запуск локально

Достаточно открыть `index.html` в браузере. Либо поднять локальный сервер:

```bash
python3 -m http.server 8000
# затем откройте http://localhost:8000
```

## Кастомизация

Цвета, радиусы и типографика вынесены в CSS-переменные в начале `assets/css/styles.css`
(блоки `:root`, `:root[data-theme="dark"]`, `:root[data-theme="light"]`).
